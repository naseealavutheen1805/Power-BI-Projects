# E-Commerce Sales Analysis Dashboard

## Overview
This project analyzes global e-commerce sales performance using Excel. The dataset was cleaned and explored using Pivot Tables, and an interactive dashboard was created to visualize sales, profit, quantity, and market performance across countries, categories, ship modes, and regions.

## Key Metrics
- **Total Countries:** 147  
- **Total Sales:** 12.64M  
- **Total Profit:** 1.47M  
- **Total Quantity Sold:** 178K  
- **Total Shipping Cost:** 1.35M  

## What I Did
- Cleaned and prepared the e-commerce dataset   
- Analyzed sales and profit across:
  - Country  
  - Category  
  - State  
  - Market  
  - Region  
  - Ship Mode  
- Built an interactive Excel dashboard with filters for Categories and Market

## Insights
- **Technology** category generated the highest sales  
- **United States** leads in total sales contribution  
- **Standard Class** accounts for the majority of shipments  
- **APAC** and **US** markets show strong sales performance  
- **Central and South regions** lead in regional sales  
- Top-performing states include **England**, **California**, and **Ile-de-France**

## Dashboard Visuals
- Sales by Country  
- Sales by Ship Mode  
- Sales by State  
- Sales by Category  
- Sales by Market  
- Sales by Region

This analysis provides a clear view of global e-commerce performance, helping identify strong markets, profitable categories, and top-selling countries to support better business decisions.
